## What I Learned today:

### How to Create a GitHub Repository
- Login to GitHub
- Click on the "New" button
- Enter repository name and description
- Select Public repository
- Click on "Create Repository"

### How to Run JavaScript Program
- Open terminal in Codespaces
- Type the command: node filename.js
- Output is displayed in the terminal

### How to commit and push code
- How to add files using git add
- How to commit changes using git commit
- How to push code to GitHub using git push
